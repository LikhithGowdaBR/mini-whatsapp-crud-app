# 💬 Mini WhatsApp CRUD App

A simple WhatsApp-inspired CRUD application built using Node.js, Express.js, MongoDB, Mongoose, and EJS. This project demonstrates how to perform Create, Read, Update, and Delete (CRUD) operations using a MongoDB database.

## 🚀 Features

- ➕ Create a new chat
- 📄 View all chats
- ✏️ Edit chat messages
- 🗑️ Delete chats
- 💾 Store chat data in MongoDB
- 🎨 Simple WhatsApp-inspired user interface

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- HTML
- CSS
- Method Override

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/LikhithGowdaBR/mini-whatsapp-crud-app.git
```

2. Go to the project folder

```bash
cd mini-whatsapp-crud-app
```

3. Install dependencies

```bash
npm install
```

4. Start MongoDB

Make sure MongoDB is running on:

```
mongodb://127.0.0.1:27017
```

5. Run the project

```bash
node index.js
```

or

```bash
nodemon index.js
```

6. Open your browser

```
http://localhost:8080/chats
```
